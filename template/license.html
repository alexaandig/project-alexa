Icons

Font icons are sourced from Unicons. You can check out the Unicons license here. To use a Unicon icon, please visit the Unicons library and select the icon you wish to use. Once you have selected the icon, you can find its unicode symbol on the right sidebar next to the icon's name. Simply click and copy the unicode symbol to use the icon.

If you want to replace an icon in a Unicons component, you can do so by pasting the icon's unicode symbol into the text field in the component properties.

SVG icons are from Flaticons and more icons from the set can be reached here.
